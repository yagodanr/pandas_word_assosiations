# pandas_word_assosiations

## Create venv and install packages:

Create virtual environment:
```Bash
python -m venv venv
```

Activate virtual environment:
* Linux
    ```Bash
    source venv/bin/activate
    ```
* Windows
    ```Bash
    . venv/Scripts/activate
    ```
* Mac
    ```Bash
    source venv/bin/activate
    ```

Install packages:
```Bash
pip install -r requirements.txt
```
